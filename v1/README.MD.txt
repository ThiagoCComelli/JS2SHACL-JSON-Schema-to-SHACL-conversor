There are 3 types of arrays:

<!-- rule 4 -->
1) Generic Array: Only type Array, accepts any type of items. => {type:array}

<!-- rule 5 -->
2) List validation array: accepts any number of elements of a specific type. Need a type array and items with the type. => {type:array,items:{type:datatype}}

<!-- rule 6 -->
3) JSON tuple array: collection of items, Generic array + List validation array.
